# The Shadow of Lillya - Manuscript Files

This directory contains all files related to *The Shadow of Lillya* by Audrey Berger Welz.

## Directory Structure

```
Shadow_of_Lillya/
├── edited_version/           # ~100 pages sent to Tyson Cornell at Rare Bird Books
│   ├── Shadow_of_Lillya_Edited.docx (original)
│   ├── Shadow_of_Lillya_Edited.pdf
│   ├── Shadow_of_Lillya_Edited.md
│   └── README.md
├── unedited_material/        # ~50 pages of additional unedited material
│   ├── Shadow_of_Lillya_Unedited.docx (original)
│   ├── Shadow_of_Lillya_Unedited.pdf
│   ├── Shadow_of_Lillya_Unedited.md
│   └── README.md
├── character_notes.md        # Character development notes
├── plot_outline.md           # Story structure and planned developments
├── notes.md                  # Any additional notes or outlines
└── README.md
```

## File Organization

### Edited Version (~100 pages)
The version sent to **Tyson Cornell** at Rare Bird Books. This represents Audrey's more polished work.

**Location:** `edited_version/` subdirectory

### Unedited Material (~50 pages)
Additional material that wasn't included in the edited version sent to the editor.

**Location:** `unedited_material/` subdirectory

## File Formats

For each version, maintain:
- **Word (.doc/.docx)** - Original source files
- **PDF** - For beautiful display in the web interface
- **Markdown (.md)** - For LLM processing and analysis
- **Text (.txt)** - Plain text for maximum compatibility (optional)

## Notes

- Keep the original Word documents as your source material
- The edited version represents Audrey's more refined work
- The unedited material may contain important character development or plot elements
- Both versions will be used as context for LLM completion attempts
- Markdown versions are optimized for LLM analysis and completion work
