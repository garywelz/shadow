# Circus of the Queens - Manuscript Files

This directory contains all files related to *Circus of the Queens* by Audrey Berger Welz.

## Expected Files

### Primary Formats
- `Circus_of_the_Queens.pdf` - PDF version for display
- `Circus_of_the_Queens.md` - Markdown version for LLM processing
- `Circus_of_the_Queens.txt` - Plain text version (optional)

### Source Files
- `Circus_Of_The_Queens_for_Rare_Bird.docx` - Original Word document (or your original filename)

## File Organization

Place all versions of *Circus of the Queens* in this directory. The application will use these files as reference material for completing *The Shadow of Lillya*.

## Notes

- Keep the original Word document as your source material
- PDF is for beautiful display in the web interface
- Markdown is optimized for LLM analysis and completion work
- Text files provide maximum compatibility
